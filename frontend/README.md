# An Homes - Next.js Frontend

## Setup
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Tech Stack
- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Cloudinary (images)

## Structure
- `app/(public)/` - Public pages
- `app/(admin)/` - Admin CMS
- `app/api/` - API routes
- `components/` - Reusable components
- `lib/` - Utilities and API client
- `types/` - TypeScript interfaces
